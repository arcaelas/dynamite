Enter @arcaelas/dynamite: The Game Changer

@arcaelas/dynamite isn't just another ORM. It's a decorator-first, zero-boilerplate solution that transforms DynamoDB from a beast into your best friend.

Here's what convinced me in literally 30 seconds:

import { connect, Table, Index, CreatedAt } from "@arcaelas/dynamite";

connect({ region: "us-east-1" });

class User extends Table {
  @Index() declare id: string;
  @CreatedAt() declare created: string;
  declare name: string;
  declare email: string;
}

// That's it. Your table is ready.
const user = await User.create({ 
  id: "u1", 
  name: "John Doe",
  email: "john@example.com" 
});

13 lines. Table defined. CRUD operations ready. Type safety included.

No CloudFormation. No manual table creation. No tears.

Getting Started in 60 Seconds

Step 1: Install the package

npm i @arcaelas/dynamite
npm i @aws-sdk/client-dynamodb @aws-sdk/util-dynamodb pluralize

Step 2: Define your model

import { connect, Table, Index } from "@arcaelas/dynamite";

connect({ region: "us-east-1" });

class Task extends Table {
  @Index() declare id: string;
  declare title: string;
  declare completed: boolean;
}

Step 3: Start building

const task = await Task.create({
  id: "task-001",
  title: "Try @arcaelas/dynamite",
  completed: false
});
